⚠️ Problema típico: el nombre y el cultivo se escriben como texto libre → "carlos mendoza" vs "Carlos Mendoza", "Palto" vs "Palta " (con espacio). Datos duplicados e inconsistentes que rompen los reportes.

✅ Ventajas

  • +Costo casi cero (ya tienen Microsoft 365)
  • +Rápido de montar — ideal para validar el MVP
  • +El cliente puede abrir y ver los datos en Excel
  • +Sin servidores que administrar

⚠️ Limitaciones

  • −Sin integridad: datos duplicados e inconsistentes
  • −No hay relaciones reales entre hojas
  • −Lento y frágil con miles de registros
  • −Conflictos si dos técnicos escriben a la vez
  • −Sin modo offline robusto ni control de permisos fino
✓ EN PRODUCCIÓN HOY

Supabase · PostgreSQL + Storage

La app está conectada a una base de datos relacional en la nube (Supabase). Captura offline en una base local SQLite y sincroniza sola al reconectar; las fotos van a Storage. Validado en dispositivo real.

-- El catálogo que gestiona el administrador (tu pedido del código de cultivo) CREATE TABLE cultivos ( codigo VARCHAR(10) PRIMARY KEY, -- CUL-001 nombre VARCHAR(60) NOT NULL UNIQUE, activo BOOLEAN DEFAULT true ); CREATE TABLE visitas ( id_visita VARCHAR(20) PRIMARY KEY, id_productor VARCHAR(20) REFERENCES productores(id_productor), cod_cultivo VARCHAR(10) REFERENCES cultivos(codigo) -- ← relación, no texto );

✅ Si el admin renombra "Palto" → "Palta Hass", cambia en un solo lugar y se refleja en todas las visitas. Imposible registrar un cultivo que no existe en el catálogo.

✅ Ventajas

  • +Integridad referencial: cero datos huérfanos o duplicados
  • +El admin gestiona catálogos (cultivos, técnicos) por código
  • +Modo offline real + sincronización confiable
  • +Roles y permisos (campo / supervisor / admin)
  • +Escala a millones de registros sin perder velocidad
  • +Reportes y exportación a Excel/PDF cuando se requiera

⚠️ A considerar

  • −Costo mensual de hosting (bajo: ~USD 0–25/mes inicial)
  • −Migración de los datos existentes del Excel (one-time)
  • −El cliente ya no edita la "BD" directo en Excel
